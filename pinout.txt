================================================================================
ELECTRICAL CHARACTERISTICS
================================================================================

OUTPUT DRIVE STRENGTH: 8mA (standard for most signals)
INPUT PULL MODE:
  - Pull-up enabled for: reset_button_n, hall_sensors, I2C lines, can_rx, leds
  - No pull for: clk, uart_rx, uart_tx, gate_signals

I/O STANDARDS:
  - 1.8V signals: BANK_VCCIO=1.8
  - 3.3V signals: IO_TYPE=LVCMOS33, BANK_VCCIO=3.3

SPECIAL CONFIGURATIONS:
  - SDA: OPEN_DRAIN=ON (required for I2C bidirectional communication)
  - All signals: PCI_CLAMP=OFF where specified

================================================================================
BLDC MOTOR CONTROL MAPPING
================================================================================

Gate Driver Connection (typical 3-phase BLDC):
  Phase A High-Side: gate_signals[0] (G1)
  Phase A Low-Side:  gate_signals[1] (G2)
  Phase B High-Side: gate_signals[2] (G3)
  Phase B Low-Side:  gate_signals[3] (G4)
  Phase C High-Side: gate_signals[4] (G5)
  Phase C Low-Side:  gate_signals[5] (G6)

Hall Sensor States (typical sequence):
  State 1: hall_sensors = 3'b001
  State 2: hall_sensors = 3'b011
  State 3: hall_sensors = 3'b010
  State 4: hall_sensors = 3'b110
  State 5: hall_sensors = 3'b100
  State 6: hall_sensors = 3'b101

================================================================================
INTERFACE REQUIREMENTS
================================================================================

I2C INTERFACE:
  - External pull-up resistors : 4.7kΩ on both SDA and SCL
  - Max clock frequency: 400kHz (Fast Mode) or 100kHz (Standard Mode)
  - Voltage level: 1.8V

CAN BUS INTERFACE:(issues)
  - Requires external CAN transceiver MCP2551, SN65HVD230, TJA1050
  - Transceiver should match FPGA voltage level (1.8V logic side)
  - CAN bus requires 120Ω termination resistors at both ends

GATE DRIVERS:
  - External gate driver IC required e.g., IR2104, IR2110, DRV8302
  - Voltage level: 3.3V compatible
  - Ensure proper dead-time to prevent shoot-through

HALL SENSORS:
  - Direct connection if 1.8V output
  - Use level shifter if sensors output 3.3V or 5V
  - Sensors should have open-collector or open-drain outputs
